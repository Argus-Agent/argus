# CodeAgent UI 快速使用说明
## 🚀 快速启动

### 1. 激活环境
```bash
conda activate dl
```

### 2. 启动UI
```bash
python run_ui.py
```

## 🎮 使用流程

### 基本操作
1. **发送任务**: 在底部输入框输入任务描述，点击"发送"
2. **查看AI回复**: AI会流式输出Markdown内容
3. **代码执行**: 代码块会自动高亮，显示"确认执行"/"取消"按钮
4. **查看结果**: 执行结果显示在代码块下方

### 功能特性
- ✅ **流式输出**: 实时显示AI思考和回复过程
- ✅ **代码高亮**: 自动识别和语法高亮代码块
- ✅ **权限控制**: 每个代码块都需要用户确认
- ✅ **结果展示**: 支持文本和图片结果
- ✅ **实时控制**: 可随时停止任务执行

## 📋 文件说明

- `code_agent_ui.py` - 主UI界面
- `run_ui.py` - UI启动脚本
- `test_simple.py` - 基础功能测试
- `.env` - API配置文件

## ⚠️ 注意事项

1. **API服务**: 确保LM Studio或其他API服务在 `http://127.0.0.1:1234/v1` 运行
2. **环境依赖**: 需要conda环境dl，已安装flet、markdown等库
3. **权限机制**: 所有代码执行都需要用户明确确认

## 🐛 故障排除

如果遇到问题：
1. 检查Python环境: `conda activate dl`
2. 测试基础功能: `python test_simple.py`
3. 检查API配置: 查看`.env`文件内容
4. 重启UI: 关闭窗口后重新运行`python run_ui.py`
